# ✨ Word Style Changer (Noun Game)

## 🧠 Description
Word Style Changer is a creative JavaScript project that displays random words (names, nouns, titles).  
Each word changes with a different style such as color, font weight, and text transformation.

---

## 🎯 Features
- Random word generator (e.g. Rihab, Queen, DIVA)
- Dynamic background highlight colors
- Font style changes (bold, italic, etc.)
- Text transformations (uppercase, lowercase, capitalize)
- Press any key or Enter to change word
- Smooth animations

---

## 🎮 How to Use
- Press any key → change word + style
- Press Enter → also change word + style

---

- JavaScript (DOM manipulation)
